IoT Data Server

A comprehensive platform for collecting, storing, and visualizing IoT device data

Version 1.0

Platform Overview

IoT Data Server là một nền tảng quản lý dữ liệu IoT đa chức năng, được thiết kế để giúp thu thập, lưu trữ và hiển thị dữ liệu từ các thiết bị IoT một cách dễ dàng và hiệu quả.

Nền tảng này hỗ trợ cả giao thức MQTT và HTTP, cho phép tích hợp với nhiều loại thiết bị IoT khác nhau, từ vi điều khiển Arduino, ESP8266, ESP32 đến các thiết bị Raspberry Pi và các cảm biến thông minh.

Tính năng chính

  • Thu thập dữ liệu qua MQTT và HTTP API
  • Quản lý thiết bị và chủ đề (topics) linh hoạt
  • Hiển thị dữ liệu telemetry theo thời gian thực
  • Xuất dữ liệu dưới định dạng CSV
  • Xác thực và phân quyền qua API keys
